# Expense Tracker — DevOps Portfolio Project

Flask REST API deployed on Kubernetes locally (Rancher Desktop) and on Azure (AKS) with Terraform + CI/CD.
Designed to be demoable and destroyable (minimal cloud cost).

## Repo structure
- `app/` — Flask API code
- `k8s/` — Kubernetes manifests
- `terraform/` — IaC for Azure
- `.github/workflows/` — CI/CD pipelines
- `docs/` — documentation & diagrams
- `scripts/` — helper scripts
